# The-Atlas-Search
A mini search engine of Countries which gives you details regarding Capital, Population, Region, Languages, Traditional dishes and clothing.
# Features ✨
* Search bar - Search input with enter button
* Real-time date - Fetches live and accurate data
* UI Rendering - Updates the page
* Error handling - Shows countries that can not be found
# TechStack ⚒️
* HTML5 - For the semantic structure of the atlas
* Tailwind CSS - For a modern, clean layout and responsive design
* JavaScript - For handling the logic, API calls, and DOM updates
* REST Countries API - As the primary data source for global information
